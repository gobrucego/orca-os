## What to Extract from Reviews

When reading these reviews, note:

### Aesthetic Language
- "Sophisticated" - What creates that quality?
- "Warm" vs "Cold" - How fonts convey temperature
- "Elegant" vs "Casual" - Personality spectrum
- "Refined" - What details create refinement?

### Pairing Principles
- Why certain fonts work together
- Contrast vs. harmony
- Display + body combinations
- What creates visual rhythm

### Context & Mood
- Editorial vs. Technical
- Formal vs. Approachable
- Timeless vs. Modern
- Authoritative vs. Friendly

### Technical Excellence
- Proportion and balance
- Letterform quality
- Spacing and rhythm
- Optical adjustments

## Using Reviews in Design Decisions

**Before choosing typography:**

1. **Read the review** of fonts you're considering
2. **Understand the personality** each font creates
3. **Match to context:**
   - Need authority? → Domaine Display
   - Need readability? → GT Pantheon
   - Need clarity? → Supreme LL
4. **Apply aesthetic principles** from the review

**Example workflow:**
```
Designing protocol page hero:
1. Read Domaine review
2. Understand: "sophisticated, authoritative, elegant"
3. Decision: Use Domaine Sans Display 48px
4. Why: Creates authority for medical protocol
5. Pair with: GT Pantheon body (review says they complement)
```

## Typography as System, Not Just Fonts

Reviews teach that great typography is about:

### Hierarchy
- How sizes create structure
- Visual rhythm through scale
- What draws the eye

### Personality
- What feeling does the font create?
- Does it match the content's tone?
- Warm vs. sophisticated vs. technical

### Pairing
- Display + body combination
- Contrast without conflict
- Creating visual interest

### Refinement
- Subtle details that add elegance
- What separates good from great
- Optical adjustments matter
